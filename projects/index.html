recurse-faces

recurse-faces: Web app that turns the Recurse Center's attendee/alumni directory data into a "learn-that-person's-name" game. Contributed to many bug fixes and front-end feature requests. 'Dockerized' the whole project so new people wanting to contribute to it can just clone the repository, run the 'docker-compose up' command and have all the development environments and dependencies running inside docker containers. Technologies used:

  • React
  • Python/Flask for api
  • PostgreSQL
  • Docker/Compose for dev environment consistency among contributors
  • Hosted on Heroku

6-Key-QWERTY

6-Key-QWERTY: Six key portable chording keyboard that spatially maps QWERTY to six keys. Type while standing, pacing, lying down, etc. This project consists of two main parts:

  • A web-app to practice the new keyboard layout
  • A physical keyboard that uses an Arduino micro-controller and 6 push buttons. Uses bluetooth radio and the HID protocol to register as a 'standard' wireless keyboard/device on all computers/phones/devices without requiring any special drivers.

Deployed the server code to a local Raspberry Pi running RancherOS so that the chat room can be reached locally at all times and she does not need to run it from her computer.
